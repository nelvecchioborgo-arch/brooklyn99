Smart Agenda - Avvio ambienti
==============================

Unificato (backend + frontend):
  dev.bat    -> DB volatile locale (dev)
  test.bat   -> DB persistente locale (test)
  prod.bat   -> DB su NAS (prod)

Solo backend (senza frontend):
  set APP_ENV=dev  && uvicorn backend.main:app --reload
  set APP_ENV=test && uvicorn backend.main:app --reload
  set APP_ENV=prod && uvicorn backend.main:app

Solo frontend:
  cd frontend && npm run dev
  (VITE_API_BASE_URL ereditato dall'ambiente o da frontend/.env)

Build produzione:
  set VITE_API_BASE_URL=http://192.168.11.20:8000
  cd frontend && npm run build
